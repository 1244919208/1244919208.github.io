什么是 CI

CI(Continuous Integration) 翻译为持续集成。Travis CI 是一个提供持续集成功能的平台，Github 可关联 Travis CI，当 GitHub 仓库有代码 push 时，会推送通知到 Travis CI，根据设置的脚本运行指定任务。
